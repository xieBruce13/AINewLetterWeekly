# DESIGN.md — AI Weekly Briefing **v4**

**来源：** 以 `npx getdesign@latest add elevenlabs` 安装的 ElevenLabs 设计说明为起点，按 **A4 打印周报** 收敛为可执行令牌（本文件为唯一权威；`tools/convert_to_pdf.py` 须与之对齐）。

**气质：** 近白画布、暖中性灰、**Inter 全站**（不引入 Waldenburg 等展示字体以免授权与加载问题）。ElevenLabs 站点强调「留白 + 轻阴影 + 正文字距」；周报为 **扁平印刷品**，**不使用 box-shadow**，用 **细边框与圆角** 区分层级。

**v4 核心约束（可读性）：**

1. **字号收紧：** 正文 **10.5pt** 为基准；标题与正文差距控制在约 **+1～+6pt**，避免出现 32pt 标题与 7.5pt 脚注同框的跳变。
2. **层级靠字重与留白：** 同字号下用 400/500/700 区分；**TL;DR** 与正文同字号，仅 **加粗**。
3. **TL;DR 文案：** 每条主表第一行「总结」列只写 **一句可扫读的摘要**（建议以 `**TL;DR：**` 开头），不写长段落。

---

## 1. 颜色（暖中性 + 单色强调）

| Token | Hex | 用途 |
|-------|-----|------|
| `ink` | `#111111` | 主文、标题、左侧 rail、TL;DR 标签 |
| `text-secondary` | `#4e4e4e` | 副标题、说明 |
| `text-muted` | `#777169` | 弱化、图注 |
| `surface-page` | `#f5f5f5` | 页背景 |
| `surface-card` | `#ffffff` | 卡片 |
| `surface-sand` | `#ebe8e4` | 表头、左侧标签列、TL;DR 行底 |
| `surface-fog` | `#f0f0ee` | 简讯 tray |
| `surface-zebra` | `#e8e6e1` | 简讯斑马纹 |
| `warm-wash` | `rgba(245,242,239,0.65)` | 引用块 |
| `surface-inverse` | `#1c1c1c` | 参考来源页脚 |
| `border-subtle` | `#e5e5e5` | 分割线、表框 |
| `link-blue` | `#2563eb` | 正文链接 |

**不要：** 冷灰、霓虹色、大面积渐变背景（与印刷可读性冲突的装饰）。

---

## 2. 字号（打印 / HTML 一致）

| 角色 | 字号 | 字重 | 说明 |
|------|------|------|------|
| 标题 h1 | 17pt | 700 | 报头 |
| 副标题 | 10.5pt | 500 | 日期 / 主题 |
| 模块 h2 | 12.5pt | 700 | 左 4px `ink` rail |
| 条目 h3 | 11pt | 700 | 卡片标题 |
| 正文 / 表内文 / deck 正文 | **10.5pt** | 400–500 | 主阅读层 |
| TL;DR 右格 | **10.5pt** | **700** | 与正文同尺寸，仅加粗 |
| 小标签 / kicker | 9.5pt | 600–700 | 全大写拉丁标签 |
| 参考来源列表 | 8.5pt | 400 | 略小于正文，但不过小 |

正文字距：**+0.12px**（ElevenLabs 式微正字距，略增透气感）。

---

## 3. 组件（与渲染器对应）

- **标题卡：** 白底圆角卡片，`ink` 圆点 + h1。
- **本周结论 deck：** 白底 + 左侧 `ink` 粗条；标签行 9.5pt uppercase；正文 10.5pt。
- **详情表：** 首行 TL;DR：沙底 + 内容格左侧 `ink` 竖条；**总结**列自动加 `· TL;DR` 后缀（CSS）。
- **信息 2×2 表、简讯、引用、页脚：** 见 `tools/convert_to_pdf.py` 内注释。

---

## 4. 页面

- A4，`@page` 边距 14mm / 16mm。
- 内容 `max-width: 820px` 居中。
- `<mark>`：无高亮底色（与正文一致）。
